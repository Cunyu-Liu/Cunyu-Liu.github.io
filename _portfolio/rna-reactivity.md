This project studies how RNA sequence and structure encode magnesium binding and oxidation reactivity.

## Overview

This project uses machine learning and statistical modeling to understand how RNA sequence and secondary structure affect magnesium binding and oxidation behavior.

## My Role

I built the analysis platform, feature engineering pipeline, ensemble model training workflow, and downstream statistical analysis modules.

## Key Work

- Trained LightGBM and soft-voting ensemble models using sequence, structure, oxidation, and metal-binding behavior data.
- Designed custom loss functions for oxidation mutation sites.
- Used window-based feature engineering to handle variable-length RNA sequences.
- Applied Bayesian optimization for parameter search and model selection.
- Explored inverse RNA sequence generation for designing sequences with targeted low or high reactivity.
- Built statistical analysis and WebLogo workflows to study high-reactivity secondary-structure features.

## Research Focus

- How do sequence and secondary structure jointly encode RNA chemical reactivity?
- Can model-derived features produce interpretable hypotheses for RNA reactivity mechanisms?
- How can generation and statistical analysis be combined to support RNA design?

## Tech Stack

**Python, LightGBM, ensemble learning, Bayesian optimization, statistical modeling, feature engineering, WebLogo**
