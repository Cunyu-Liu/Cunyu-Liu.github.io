This project studies how RNA sequence and structure encode magnesium binding and oxidation reactivity.

**Highlights**

- Built an AI analysis platform for RNA reactivity at Westlake University.
- Trained LightGBM and soft-voting ensemble models using sequence, structure, oxidation, and metal-binding behavior data.
- Designed custom loss functions for oxidation mutation sites and window-based feature engineering for variable-length RNA sequences.
- Used Bayesian optimization for model selection and parameter tuning.
- Explored inverse RNA sequence generation for designing sequences with targeted reactivity.
- Built statistical analysis and WebLogo workflows for high-reactivity secondary-structure features.

**Tech stack:** Python, LightGBM, Bayesian optimization, statistical modeling, WebLogo.
