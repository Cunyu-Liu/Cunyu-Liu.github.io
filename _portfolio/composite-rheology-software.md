- Built the frontend with Vue and the backend with Django.
- Implemented chart visualization with ECharts, including pie charts, line charts, and bar charts.
- Integrated MySQL data operations and automated PDF report generation.
- Participated in unit testing, integration testing, and deployment.
- Delivered software copyright and deployment in a national key laboratory.

**Tech stack:** Vue, Django, ECharts, MySQL, PDF generation.
