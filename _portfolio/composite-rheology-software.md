Led a 7-member student team to build a scientific analysis web platform for composite material rheology and constitutive modeling.

## Overview

This project built a web-based scientific software system for composite material rheological and constitutive analysis. The platform supports data management, analysis visualization, and automated report generation.

## My Role

I served as project lead and coordinated a team of 6 graduate students and 1 undergraduate student across frontend, backend, visualization, testing, and deployment.

## Key Work

- Designed and implemented frontend pages with Vue.
- Built backend analysis and data interfaces with Django.
- Implemented ECharts visualizations, including pie charts, line charts, and bar charts.
- Integrated MySQL data operations for analysis workflows.
- Added automated PDF report generation from backend analysis results.
- Participated in unit testing, integration testing, and deployment.

## Impact

- Delivered a deployed system for a national key laboratory.
- Obtained software copyright.
- Built a complete full-stack scientific software workflow from data analysis to report generation.

## Tech Stack

**Vue, Django, ECharts, MySQL, PDF generation, unit testing, integration testing**
