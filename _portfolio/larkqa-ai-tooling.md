At ByteDance LarkQA, I worked on AI engineering systems that automate test case polishing, defect coverage review, and platform testing workflows.

**Highlights**

- Built BBC/BBQ test case polishing workflows with Coze Bot API, Aily asynchronous runs, Fornax GPT, multimodal prompts, and Spring Boot + Redis queues.
- Designed quantitative adoption metrics for polished test steps and scheduled jobs for data-quality maintenance.
- Built RAG-based defect automation coverage review with BBQ/BBC dual recall, multimodal information completeness checks, and confidence scoring.
- Integrated review results into Feishu Bitable with a structured output schema.
- Built Bits/BOSS testing tools, BOSS CLI, and OpenClaw Skill Gateway for tenant creation, asset unsubscribe, usage query/reporting, and mock-time scenarios.

**Tech stack:** Java, Spring Boot, Python, Node.js, TypeScript, Redis, Coze API, Aily API, Fornax GPT, Viking DB, Milvus, MCP SDK.
